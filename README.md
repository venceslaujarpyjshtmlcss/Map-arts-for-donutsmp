# Map-arts-for-donutsmp
Random map arts for donut smp!

you can use these public map arts in donut smp or another Minecraft servers

# Litematica

you need litematica to build the map arts in the word, the material list is included in the map art file

Download here : https://modrinth.com/mod/litematica

# How to load the schematic of the map art?

to load the schematic of the map art you need to move the .nbt file of the map art to the "schmatics" folder im the Minecraft folder
